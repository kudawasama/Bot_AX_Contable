# Skill: Verify

Propósito:
Comprobar automáticamente que las propuestas no rompen el código y son seguras de aplicar.

Comportamiento:
- Ejecutar lint/compilación básica y reportar fallos.
- Si existe soporte para tests, indicar comandos para correrlos.
- Clasificar issues en pass/warn/fail y marcar blocking si aplica.

Salida:
- `results`, `summary`, `blocking`.
